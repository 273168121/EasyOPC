
## Easy OPC
Eeasy OPC is a .NET wrapper over the [OPC Foundation UA .NET](https://github.com/OPCFoundation/UA-.NET) library.


### Prerequiesites
- A 2048 bits key size valid certificate, for the "CurrentUser" store.
